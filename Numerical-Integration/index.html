    INTEGRASI NUMERIK¶

    Integral dibagi menjadi 2, yaitu integral tentu dan integral tak tentu

    • INTEGRAL TAK TENTU¶

    Integral tak tentu merupakan suatu fungsi yang memiliki nilai berbeda satu sama lain tergantung dengan nilai konstanta yang dimilikinya.

    \[\int xdx=\dfrac{x^{2}}{2}+c\]

    Dimana:

    o C adalah sebuah nilai konstanta.

    • INTEGRAL TENTU¶

    Integral tentu merupakan suatu fungsi untuk memperolah hasil berupa nilai murni.

    \[I=\int ^{b}_{a}f\left( x\right) dx\]

    Dimana:

    o a dan b adalah batas – batas integrasi fungsi yang ada.

    o f adalah sebuah simbol untuk fungsi yang bentuknya dapat diberikan secara eksplisit berupa persamaan ataupun secara empiric berupa table yang memiliki nilai.

    o Nilai integral tentu merupakan sebuah luas daerah dibawah kurva y = f(x) yang dibatasi oleh garis x = a dan garis x = b.

    \[\int ^{b}_{a}f\left( x\right) dx= F\left( x\right) | _{a}^{b}= F(b) – F(a)\]

    Dimana F(x) adalah fungsi kontinu,

    o Banyak integral tentu yang dapat dihitung dengan menggunakan rumus tersebut, akan tetapi tidak sedikit integral tentu yang tidak dapat dihitung dengan rumus tersebut karena integral f(x) tidak mempunyai antiderivative yang dinyatakan dalam fungsi – fungsi elementer.

    Integrasi Numerik / Numerical Integration¶

    Integrasi numerik merupakan suatu ilmu yang digunakan untuk mendapatkna nilai – nilai hampiran untuk integral tentu yang tidak dapat diselesaikan secara analitik.

    KENAPA ADA ILMU INTEGRASI NUMERIK?¶

    Karena tidak semua permasalahan dapat diselesaikan dengan integral yang biasa atapun dihitung secara analitik, oleh karena itu dibuat metode – metode integrasi untuk menyelesaikan masalah tersebut.

    APA ITU INTEGRASI NUMERIK?¶

    Integrasi numerik merupakan penjumlahan suatu fungsi untuk mendapatkan semua nilai atau luas daerah dibawah kurva tertentu.

    • Metode Integrasi Tapezoid

    Metode trapezoid adalah salah satu metode dalam integrasi numerik menggunakan pendekatan penjumlahan beberapa segmen untuk mencari suatu nilai. Pada metode ini, terdapat sautu kurva yang kemudian diatrik garis lurus untuk menghasilkan suatu bentuk luar trapesium. Panjang satu pias trapesium adalah dari x = x0 sampai x = x1.

    Kemudian luas suatu trapesium sendiri dapat didefinisikan seperti integral fungsi berikut:

    \[\int ^{x}_{x_{0}}f\left( x\right) dx\approx \dfrac{n}{2}\left[ f\left( x_{0}\right) +f\left( x_{1}\right) \right]\]

    Fungsi diatas dapat disebut sebagai kaidah trapezium.

    Apabila antara selang [a,b] dibagi ataso atas n buah pias trapezium, kaidah integrasi yang diperoleh adalah kaidah trapezium gabungan (composite trapezoidal’s rule) sebagai berikut:

    \[\int ^{b}_{a}f\left( x\right) dx\approx \int ^{x\cdot }_{x_{0}}f\left( x\right) dx+\int ^{x_{2}}_{x.}f\left( x\right) dx+\ldots t_{x}\int ^{x_{n}}_{n-1}f\left( x\right) dx\]
    \[\int ^{b}_{a}f\left( x\right) dx\approx \dfrac{h}{2}\left[ f\left( x_{0}\right) +\int \left( x_{1}\right) \right] +\dfrac{h}{2}\left[ f\left( x_{1}\right) +f\left( x_{2}\right) \right] +\ldots + \dfrac{h}{2}\left[ f\left( x_{n-1}\right) +f\left( x_{n}\right) \right]\]
    \[\int ^{b}_{a}f\left( x\right) dx\approx \dfrac{h}{2}[ f\left( x_{0}\right) +2f\left( x_{1}\right) +2f\left( x_{2}\right) +\ldots + 2f\left( x_{n}-1\right) +f\left( x_{n}\right) ]\]
    \[\int ^{b}_{a}f\left( x\right) dx\approx \dfrac{h}{2}\left( f_{0}+2f_{1}+2f_{2}+\ldots +2f_{n-i}+f_{n}\right)\]
    \[\int ^{b}_{a}f\left( x\right) dx\approx \dfrac{h}{2}\left( f_{0}+2\Sigma _{r=1}^{n-1}f_{r}+f_{n}\right) dengan \int _{r}=f\left( x_{r}\right) r=0,1,2,\ldots ,n\]

    Kemudian untuk menentukan galat pada dengan metode trapezoid dapat menggunakan rumus berikut:

    \[E_{tot}=-\dfrac{h^{2}}{12}\left( b-a\right) f''\left( t\right) ,0 <t <h\]

    Adapun Algoritma untuk metode Trapezoid adalah sebagai berikut:

    1. Mendefinisikan fungsi pada integral yang sudah ditentukan
    2. Menambahkan batas atas dan batas bawah pada integrasi yang akan dilakukan
    3. Menentukan nilai n pada integrasi
    4. Menghitung nilai h dengan rumus h = (b-a)/2
    5. Melakukan perhitungan dengan perulangan sampai mengeluarkan hasil

    Berikut merupakan penerapan metode Trapezoid ke dalam bentuk code pyhton:

    from math import sin, pi

bawah = 0
atas = pi/2
n = 1
h = ((atas-bawah)/ (2**n))
f = lambda x: sin(x)

def trapezoid (bawah, atas, f, h, n):
    S = 0.5 * (f(bawah) + f(atas))
    for i in range(0,n+1):
        x = bawah + i*h
        S+= f(x)
    totalS = h * S
    return totalS


print(trapezoid(bawah,atas,f,h,n))
    • Metode Integrasi Romberg

    Metode romberg adalah metode integrasi yang didasarkan pada aturan metode trapezoid dan error calculation sehingga dapat menghasilkan nilai integral dengan tingkat presisi yang tinggi. Metode ini didasarkan pada perluasan ekstrapolasi Richardson untuk memperoleh nilai integrasi yang semakin baik.

    Tabel Romberg:

    R(0,0)
    R(1,0) R(1,1)
    R(2,0) R(2,1) R(2,2)
    R(3,0) R(3,1) R(3,2) R(3,3)

    Rumus:

    \[R\left( 0,0\right) =\dfrac{b-a}{2}\left[ f\left( a\right) +f\left( b\right) \right] \]
    \[R\left( n,0\right) =\dfrac{1}{2}R\left( n-1\right) +h\sum ^{2^{n-1}}_{k=1}f\left( a+\left( 2k-1\right) h\right) \]
    \[R\left( n,m\right) =\dfrac{1}{4^{m}-1}\left[ 4^{n}\times R,n,m-1) -R\left( n-1,m-1\right) \right] n\geq 1,m\geq 1,n <n\]

    Adapun Algoritma untuk metode Trapezoid adalah sebagai berikut:

    1. Menentukan nilai h
    2. Mencari nilai R(0,0) - R(n,m)
    3. Melakukan perulangan sebnyak n
    4. Mendapatkan hasil

    Berikut merupakan penerapan metode Romberg dalam bentuk code pyhton:

    def romberg(f,a,b,n):
hasil=[]
h = (b−a)
hasil.append([(h/2.0)∗(f(a)+f(b))])
for i in range(1,n+1):
    h = h/2.
    sum = 0
    for k in range(1,2∗∗i ,2):
        sum = sum + f(a+k∗h)
        hasilSementara = [0.5∗hasil[ i−1][0] + sum∗h]
        for j in range(1,i+1):
        rij = hasilSementara[j−1] + (hasilSementara[j−1]−hasil[i−1][j−1])/(4.∗∗j−1.)
        hasilSementara.append(rij)
    hasil.append(hasilSementara)

return hasil
    • Metode Integrasi Simpson ⅓

    Integrasi Numerik metode simpson ⅓ digunakan untuk menghitung sebuah luas suatu dengan tingkatan ketelitian lebih tinggi daripada metode Trapezoid. Pada metode ini setiap segmen atau pias bernilai genap akan dihitung nilanya untuk dievaluasi.

    Untuk rumus pada metode simpson ini didapat sebagai berikut:

    \[\int ^{b}_{a}f\left( x\right) dx=\dfrac{h}{3}\left( f_{0}+4\sum ^{n-1}_{i=1,3,5}f_{i}+2\sum ^{n-2}_{i=2,4,6}f_{i}+f_{n}\right)\]

    Adapun Algoritma untuk metode Simpson ⅓ adalah sebagai berikut:

    1. Mendefinisikan fungsi pada integral yang sudah ditentukan
    2. Menambahkan batas atas dan batas bawah pada integrasi yang akan dilakukan
    3. Memberi syarat pada saat perulangan dengan nilai n genap
    4. Menghitung nilai h
    5. Melakukan perulangan dan perhitungan
    6. Mendapatkan hasil

    Berikut merupakan penerapan metode Simpson ⅓ dalam bentuk code pyhton:

    # Mendefinisikan fungsi agar menjadi integral
def f(x):
    return 1/(1 + x**2)

def simpson(a,b,n):
    # mencari nilai h
    h = (b - a) / n

    # Mencari jumlah sementara 
    integrasi = f(a) + f(b)

    for i in range(1,n):
        k = a + i*h

        if i%2 == 0:
            integrasi = integrasi + 2 * f(k)
        else:
            integrasi = integrasi + 4 * f(k)

    # mendapatkan nilai akhir perhitungan
    integrasi = integrasi * h/3

    return integrasi

a = float(input("Masukkan Batas bawah: "))
b = float(input("Masukkan Batas atas: "))
n = int(input("Masukkan Jumlah interval n: "))
hasil = simpson(a, b, n)
print("Hasil Integrasi= %0.6f" % (hasil) )
